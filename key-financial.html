Infoscape: map key, financial

Colour

range of influence the organisation has:

  • green = local
  • blue = regional
  • red = national
  • purple = international

Size

Budget amount for an org in one year (scale is logarithmic - £0 is just visible, £1000 is small, £100,000 is fairly big)

Shape

shapes merge to reflect the degree to which the organisation is disability-led:

  • square = fully disability-led
  • square/sphere = part disability-led (% indicated by hybrid shape)
  • sphere = not disability-led

Axes

as viewed when page loads:

  • left - right (X axis) = county (coloured): Nottinghamshire, Lincolnshire, Leicestershire, High Peak, Northamptonshire
  • top - bottom (Y axis) = time established (1 grid square = 5 years, 0 at bottom)
  • front - back (Z axis) = financial year (grid = 1 year, back = previous years, front = 2002/03)

So each view basically contains 3 bits of comparative information...

  • View 1: X-Y = County / Time established / Budget
  • View 2: Y-Z = Time established / Year / Budget
  • View 3: X-Z = County / Year / Budget
